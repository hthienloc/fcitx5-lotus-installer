Cài dependencies

sudo xbps-install -Sy cmake extra-cmake-modules libfcitx5-devel libinput-devel eudev-libudev-devel gcc go gettext-devel pkg-config hicolor-icon-theme libx11-devel
2
Clone & build
git clone https://github.com/LotusInputMethod/fcitx5-lotus.git cd fcitx5-lotus mkdir build && cd build cmake -DCMAKE_INSTALL_PREFIX=/usr -DCMAKE_INSTALL_LIBDIR=/usr/lib -DINSTALL_RUNIT=ON -DRUNIT_SV_DIR=/etc/sv .. make sudo make install

🐧 NixOS Flake Method

Thêm input vào flake.nix:

inputs.fcitx5-lotus = { url = "github:LotusInputMethod/fcitx5-lotus"; inputs.nixpkgs.follows = "nixpkgs"; };

Cấu hình trong configuration.nix:

services.fcitx5-lotus = { enable = true; user = "your_username"; };

Rebuild system:

sudo nixos-rebuild switch

🔧 Sau khi cài đặt

Các bước cấu hình sau khi cài fcitx5-lotus

1
Kích hoạt fcitx5-lotus-server

Gói .deb/rpm tự động làm bước này. Build from source cần làm thủ công:

sudo systemctl enable --now fcitx5-lotus-server@$(whoami).service
2
Thiết lập biến môi trường

Thêm vào ~/.bash_profile (hoặc ~/.zprofile cho Zsh):

export GTK_IM_MODULE=fcitx export QT_IM_MODULE=fcitx export XMODIFIERS=@im=fcitx export SDL_IM_MODULE=fcitx export GLFW_IM_MODULE=ibus
3
Tắt IBus (nếu có)

Tránh xung đột với bộ gõ cũ:

killall ibus-daemon || ibus exit
4
Autostart fcitx5

Thêm vào DE/WM config:

# GNOME / KDE / Xfce / v.v... # Dùng Startup Applications để thêm Fcitx 5 # i3: thêm vào ~/.config/i3/config exec --no-startup-id fcitx5 -d # Sway: thêm vào ~/.config/sway/config exec --no-startup-id fcitx5 -d # Hyprland: thêm vào ~/.config/hypr/hyprland.conf exec-once = fcitx5 -d

🪟 Wayland Configuration

Nếu bạn dùng Wayland, cần lưu ý các điểm sau:

K
KDE Plasma

Vào System Settings → Keyboard → Virtual Keyboard → Chọn Fcitx 5

C
Chromium / Electron

Chạy với flags để bật Wayland IME:

--enable-features=UseOzonePlatform --ozone-platform=wayland --enable-wayland-ime
!
Xwayland khuyến nghị

Ngay cả khi chỉ dùng Wayland native, bật Xwayland giúp Fcitx hiển thị candidate window đúng vị trí.
